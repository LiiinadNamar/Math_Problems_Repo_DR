Disjoint vs overlapping events

Looks disjoint? A = {2,4,6}, C = {5,6}
Quick visual check can mislead: outcome 6 belongs to both sets
Wait — {6} is in both! Let's check carefully:
A = {2,4,6}, C = {5,6}
A ∩ C = {6} ≠ ∅
So actually addition fails here too.
f(A∪C) = f(A) + f(C) − f(A∩C)
= 0.491 + 0.330 − 0.170 = 0.651
Overlapping: M = {1,2,3}, N = {3,4,5}
Outcome 3 is in both → double counted
f(M) = 0.503, f(N) = 0.508
f(M) + f(N) = 1.011 — impossible!

Outcome 3 counted twice (181 throws).
Correct: f(M∪N) = 1.011 − 0.181 = 0.830
The rule: f(A∪B) = f(A) + f(B) − f(A∩B). Simple addition f(A)+f(B) is only correct when A∩B = ∅, because then f(A∩B) = 0 and no outcome is counted twice.

Kolmogorov axioms — origin and meaning

Axiom 1
P(A) ≥ 0 for every event A.
Origin: counts n(A) ≥ 0 always, so f(A) ≥ 0 always. Transparent from frequencies.
Axiom 2
P(Ω) = 1.
Origin: every throw lands in Ω, so n(Ω) = 1000 and f(Ω) = 1 always. Transparent from frequencies.
Axiom 3a
Finite additivity: A∩B = ∅ → P(A∪B) = P(A)+P(B).
Origin: disjoint events don't share throws, so counts add. Explained by counting logic.
Axiom 3b
Countable additivity (σ-additivity): for any infinite sequence of disjoint events,
P(A₁∪A₂∪…) = P(A₁)+P(A₂)+…

Origin: NOT from finite experiments. No finite experiment can exhibit ∞ disjoint events. Required for: infinite sample spaces (geometric distribution), continuity P(Aₙ)→P(A), law of large numbers, CLT.
Key distinction: Axioms 1–2 and finite additivity are suggested directly by observing frequencies in any experiment. Countable additivity is a new mathematical postulate — it gives probability its continuity and makes the theory work on infinite spaces. It cannot be verified empirically.
